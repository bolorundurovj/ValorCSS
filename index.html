Quick Start

Get up and running with ValorCSS in seconds

1. Installation

Clone the repository or download the CSS:

git clone git@github.com:bolorundurovj/ValorCSS.git

2. Include CSS

Add the stylesheet to your HTML:

<link rel="stylesheet"
  href="dist/main.min.css">

3. Start Building

Use utility classes and components:

<div class="flex items-center gap-4">
  <button class="btn btn-primary">
    Click me
  </button>
</div>

4. Build from Source

Customize and build with SCSS:

yarn install
yarn build:prod
